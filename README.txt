clib: components library

par_clib: parallel clib, components for a NoC implimentation in which connections are parallel to/from the local ports
create_noc: perl scripts for creating the noc connections.v file 
routing_tables: perl scripts for creating the routing table 
traffic: traffic patterns
_rr: stands for round robin, components that are used to impliment a round robin-based router
